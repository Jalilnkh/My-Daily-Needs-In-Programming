# Some minor info:

- 1- How to create bash script file:

        - use vim commend. if it is not installed on Linux use sudo apt-get install vim 
        - write scripts
        - To save:
                - Use escape + w + q

- 1- How to make bash script excutionable:

        - chmod +x b-script_name : make file excutable 
        - chmod 755 b-script-name : 7(111) means read, write, excute, 5(101) means read and excute

- 2 - How to run bash script, there are three way to do it:

        - bash bash-script-name
        - sh bash-script-name
        - ./bash-script-name
